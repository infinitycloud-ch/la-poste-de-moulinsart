# MCP Localization Test Tool

A standardized Model Context Protocol (MCP) tool for comprehensive iOS localization testing.

## Files Created

1. **mcp_localization_test.json** - MCP tool specification
2. **mcp_localization_test.sh** - Implementation script
3. **MCP_LOCALIZATION_README.md** - This documentation

## Quick Start

### 1. Integration with Claude

Add the MCP tool to your Claude session by referencing the JSON specification:

```bash
# The MCP tool can be loaded from:
~/moulinsart/agents/nestor/mcp_localization_test.json
```

### 2. Available Functions

#### check_localization
Comprehensive localization test suite:
```bash
./mcp_localization_test.sh check_localization /path/to/ios/project
```

#### validate_strings  
Check for hardcoded strings:
```bash
./mcp_localization_test.sh validate_strings /path/to/ios/project
```

#### verify_keys
Validate localization key consistency:
```bash
./mcp_localization_test.sh verify_keys /path/to/ios/project
```

#### generate_report
Create HTML report with auto-open:
```bash
./mcp_localization_test.sh generate_report /path/to/ios/project report.html true
```

## Example Integration

In your Claude session, you can now use:

```
check_localization project_path="/Users/myuser/MyApp" languages=["en","fr","es"]
validate_strings project_path="/Users/myuser/MyApp" file_patterns=["*.swift"]
verify_keys project_path="/Users/myuser/MyApp" base_language="en"
generate_report project_path="/Users/myuser/MyApp" auto_open=true
```

## Return Format

All functions return structured JSON with:
- `status`: "success", "warning", or "failure"
- `timestamp`: ISO 8601 timestamp
- Function-specific data fields
- Error details when applicable

## Project Structure Requirements

The tool expects iOS projects with:
- `.xcodeproj` or `.xcworkspace` files
- `*.lproj` directories for localizations
- `Localizable.strings` files
- Swift/Objective-C source files

## Scoring System

- **100%**: Perfect localization
- **80-99%**: Minor issues (warnings)
- **60-79%**: Moderate issues
- **<60%**: Critical issues (failure status)
