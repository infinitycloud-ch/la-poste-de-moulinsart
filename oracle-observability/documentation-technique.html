⚠️ Configuration Critique

SQLite WAL Mode obligatoire pour éviter les conflits de verrouillage entre serveurs concurrent. Sans cette config, les emails apparaissent/disparaissent de façon sporadique.

🔔 Système Notifications TMUX

🚨 Configuration Critique Validée

Algorithme final dans mail-server.ts ligne ~400 :

mainCommand += ` && sleep 2 && tmux send-keys -t ${sessionName}:agents.${panelNumber} C-m`;

Pourquoi : Résout les notifications "collantes" grâce à la temporisation de 2 secondes + return unique.
