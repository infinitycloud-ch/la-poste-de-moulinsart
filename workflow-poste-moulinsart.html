📧 Étape 1: Envoyer un Email

Vous envoyez un email à un agent (ex: nestor@moulinsart.local)

DE: vous@moulinsart.local À: nestor@moulinsart.local SUJET: Nouveau projet - Application Test Bonjour Nestor, Tu es NESTOR, le chef d'orchestre de ce projet. [Instructions spécifiques...]

🎣 Étape 2: Le Serveur Mail Reçoit

Le serveur mail (port 1025) reçoit l'email et déclenche automatiquement l'injection

📬 New mail for nestor@moulinsart.local 💉 Notification injectée dans tmux session: nestor

💉 Étape 3: Injection dans tmux

La notification est injectée dans la session tmux de l'agent

🔔 NOTIFICATION SYSTÈME ━━━━━━━━━━━━━━━━━━━━ 📧 Nouveau mail reçu! 📍 Structure: /projects et /agents 👥 Agents: nestor, tintin, dupont1, dupont2 📬 Mail: http://localhost:1080 💡 Pour envoyer: utilise Python SMTP ou curl Consulte ta boîte et suis les instructions. Pour voir tes emails: curl http://localhost:1080/mailbox/nestor ━━━━━━━━━━━━━━━━━━━━

🤖 Étape 4: Claude Réagit

Claude voit la notification et consulte automatiquement ses emails

⏺ Fetch(http://localhost:1080/mailbox/nestor) ⏺ Bash(curl http://localhost:1080/mailbox/nestor)

🎬 Étape 5: Claude Agit

Claude lit l'email, comprend son rôle et agit en conséquence

  • Lit "Tu es NESTOR..."
  • Comprend son rôle de chef d'orchestre
  • Envoie des emails aux autres agents
  • La chaîne continue automatiquement!

🚀 Démarrage Rapide

1️⃣ Lancer les Services

# Démarrer La Poste cd ~/moulinsart ./start-poste.sh # Les services démarrent: # - Oracle API sur port 3001 # - Mail Server sur ports 1025/1080

2️⃣ Créer les Sessions tmux

# Créer session pour Nestor tmux new-session -d -s nestor # Lancer Claude dans la session tmux send-keys -t nestor "claude" Enter

3️⃣ Envoyer un Email Test

# Utiliser le script Python python3 test-email.py # Ou avec curl/telnet sur port 1025
✅ Succès! Claude reçoit la notification et réagit automatiquement!

🔑 Points Clés

⚠️ Important: L'identité vient de L'EMAIL, pas du tmux! Le tmux ne dit PAS qui est l'agent - c'est dans l'EMAIL!
