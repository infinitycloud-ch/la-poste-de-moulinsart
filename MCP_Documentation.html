Le système MCP (Mobile Command Platform) est une plateforme de validation automatique révolutionnaire qui utilise l'intelligence artificielle pour détecter, analyser et corriger les problèmes de localisation dans les applications iOS. Développé pour l'équipe Moulinsart, il permet une validation complète sans intervention humaine.

🔄 Workflow Complet de Validation

1
Génération Vidéo Automatique
./scripts/validation_video_ai.sh - Teste 4 langues, parcourt toutes les vues
2
Analyse AI Multimodale
python3 analyze_video_with_ai.py - Détecte underscores, textes non traduits
3
Dispatch Intelligent
python3 validation_dispatcher_agent.py - Assigne les tâches aux bons agents
4
Corrections Parallèles
DUPONT1 (Swift) + DUPONT2 (Traductions) travaillent simultanément
5
Revalidation Automatique
Boucle jusqu'à score > 95/100

🛠️ Outils et Scripts Créés

📹 validation_video_ai.sh

Génère une vidéo de test multi-langues avec navigation automatique dans l'app.

  • 4 langues testées (FR, EN, DE, ES)
  • Navigation complète avec scroll
  • ~60 secondes de vidéo

🤖 analyze_video_with_ai.py

Analyse la vidéo avec un modèle multimodal pour détecter les erreurs.

  • Détection d'underscores
  • Vérification des traductions
  • Score de conformité

📧 validation_dispatcher_agent.py

Dispatche intelligemment les erreurs aux agents responsables.

  • Matrice de responsabilités
  • Instructions précises
  • Priorités automatiques
